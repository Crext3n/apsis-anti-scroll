# Apsis
## ----- EN -----
### Application Features
The application uses accessibility features to detect opened applications, scan the components on the screen, and if it detects a prohibited object, it goes back and prevents the video from playing.
After opening the accessibility settings in the interface, a password reset button appears and also displays information about whether the service is running or not.
It displays warning messages when attempting to reset the password.
There's also a switch in older Instagram versions that searches for components that work more stably in older versions.
### How it works
The application detects Isnstagram, TikTok, or YouTube from packages like "com.instagram.android" and continuously scans the screen to find the specified components, such as "com.instagram.android:id/comment_button".
Due to version differences and to ensure greater stability, it checks multiple components.

### Uygulamanın özellikleri
Uygulama erişilebilirlik özelliğini kullanarak açılan uygulamayı tespit ediyor, ekrandaki componentleri tarıyor, eğer yasaklanan bir nesne bulursa geri gidiyor ve o videoyu izlettirmiyor.
Arayüzünde erişilebilirlik ayarlarını açtıktan sonra şifre sıfırlama butonu beliriyor aynı zamanda servisin çalışığ çalışmadığıyla ilgili bilgi yazıyor.
Şifreyi sıfırlamaya çalışınca uyarı mesajları veriyor.
Aynı zamanda eski instagram sürümleri içinde bir switch bulunuyor bu sayede eski sürümde daha kararlı çalışan componenti arıyor.
